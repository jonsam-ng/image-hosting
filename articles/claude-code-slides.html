// query.ts — 真正的心脏
async function* queryLoop() {
  while (true) {
    const response = await callModel(messages);
    for (const block of response.content) {
      if (block.type === 'tool_use') yield executeTool(block);
    }
  }
}

四条设计哲学

  • 决策交给模型，约束交给工程
    没有外部任务规划器 — 模型决定做什么，工程确保只能做安全的事
  • 核心循环极简，复杂度推向外层
    queryLoop() 保持可读 — Prompt/Permission/Context/Cache 全部在外
  • 让约束塑造架构
    缓存需求驱动提示词结构，而非反过来
  • 安全是多层纵深，非单点拦截
    Prompt · Mode · Rules · Params · Path · Sandbox · MCP Policy 层层叠加
